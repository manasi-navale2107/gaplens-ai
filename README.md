# GapLens AI 🚀

GapLens AI is an AI-powered platform that analyzes resumes and identifies skill gaps for specific job roles.  
It helps users prepare for interviews by suggesting improvements and generating relevant questions.

## Features
- Resume analysis using AI
- Job-skill gap detection
- Interview preparation assistance
- User-friendly interface

## Tech Stack
Frontend: React.js, HTML, CSS, JavaScript  
Backend: Node.js, Express.js  
Authentication: JWT  
AI: Google Gemini API  
Database: MongoDB  
Version Control: Git, GitHub

## Installation
Clone the repository:
git clone https://github.com/manasi-navale2107/gaplens-ai.git
Go to project folder:
cd gaplens-ai
Install dependencies:
npm install
Run project:
npm start
